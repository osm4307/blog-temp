## 작업 진행 상황 체크 방법
1. 각 작업 항목을 완료할 때마다 체크박스에 체크
2. 커밋 메시지 규칙 준수
   - feat: 새로운 기능
   - fix: 버그 수정
   - docs: 문서 수정
   - style: 코드 포맷팅
   - refactor: 코드 리팩토링
   - test: 테스트 코드
   - chore: 빌드/설정 관련
   - perf: 성능 개선

## 주의사항
- 각 작업 단위별로 커밋하기
- 커밋 전 코드 리뷰 진행
- 테스트 코드 작성 필수
- 문서화 작업 병행
